# examples/ssr

https://vite-plugins-example-ssr-hiro18181.vercel.app/

```sh
# update dependencies
pnpm update

# development
pnpm i
pnpm dev

# build for local preview
pnpm build-preview
pnpm preview

# build for vercel edge (see misc/vercel-edge/README.md)
pnpm build-vercel-edge
pnpm release
```
